# myotis-edna-detection-probability
R code for airborne eDNA detection probability and sensitivity analysis of Myotis bats.
# Myotis eDNA Detection Probability

This repository contains the R code and supporting data used for the analyses presented in the manuscript on airborne eDNA detection probability in Myotis bats.

## Repository contents

- `Master code.R` – R script used for all statistical analyses and figure generation.
- `maternityroost_qpcr.csv` – qPCR data from maternity roost samples.
- `bunker_qpcr.csv` – qPCR data from bunker samples.
- `hibernacula_qpcr.csv` – qPCR data from hibernacula samples.

## Requirements

The analyses were performed in R using the packages:

- dplyr
- tidyr
- ggplot2
- grid
- patchwork

## Data availability

All data and R code required to reproduce the analyses are available in this repository.
